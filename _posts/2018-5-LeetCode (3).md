# LeetCode 5-25

## 关于DFS算法

> DFS算法其实就是回溯算法。
>
> 解决一个回溯问题，实际上就是一个决策树的遍历过程。其实值需要思考三个问题：
>
> 1->路径：也就是已经做出的选择。
>
> 2->选择列表：也就是你当前可以做的选择。
>
> 3->结束条件：也就是到达决策树底层，无法再做选择的条件。
>
> 代码方面，回溯算法的框架：

~~~~
result = []
def  backtrack(路径 ， 选择列表):
	if 满足结束条件：
		result.add(路径)
		return
	for 选择 in 选择列表：
		做选择
		backtrace(路径， 选择列表)
		撤销选择
~~~~

> 其核心就是for循环里边的递归，再递归调用之后撤销 选择，特别简单。
